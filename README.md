## Project setup
```
npm install

python3 -m venv ./venv
# Windows
# ./venv/Scripts/activate.bat

# Linux/Git Bash
source ./venv/bin/activate

pip install -r requirements.txt
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
